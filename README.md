# bogoscheme

A subset of [Scheme](https://www.gnu.org/software/mit-scheme/)

## Examples

```bash
# Interprets a file
$ ./bs ./examples/fibonacci.scm
0
1
1
2
3
5
55
9227465
12586269025

# REPL
$ ./bs
BogoScheme REPL (Press Ctrl-D to exit)
> (define x 10)
#u
> (print x)
10
#u
>
```
